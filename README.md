#  Cat vs Dog Image Classification Using CNN

##  Project Overview

This project presents a Deep Learning-based image classification system that utilizes a Convolutional Neural Network (CNN) to distinguish between cat and dog images. The model is developed using TensorFlow and Keras, incorporating image preprocessing, data augmentation, and feature extraction techniques to enhance classification performance and generalization.

---

##  Objectives

- Develop a CNN architecture for binary image classification.
- Automate feature extraction using deep learning.
- Improve model robustness through data augmentation.
- Evaluate model performance using classification metrics.
- Perform real-time prediction on unseen images.
